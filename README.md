- **Ultra-Pure Process Water** (Electronics & IT)

## 🎨 Design Philosophy

The user interface follows an **ultra-premium, industrial corporate aesthetic**:
- **Typography:** Outfit (Headings) and Inter (Body) for a highly legible, technical, and modern structural feel.
- **Layout:** Edge-to-edge minimalist editorial split-layouts, entirely discarding traditional boxed cards for a more expansive, authoritative presentation.
- **Glassmorphism:** Implementing floating navigation and subtle blurs matching Apple/Tesla-level presentation standards.
- **Performance:** Lightweight, dependency-free vanilla HTML, CSS, and JS. Zero bloat, ensuring maximum rendering speed.

## 🚀 Setup & Local Development

This project is built using pure, vanilla web technologies. No complex build steps or node packages are required.

1. Clone the repository:
   ```bash
   git clone https://github.com/krishanth7/venigem-tech.git
   ```
2. Navigate to the project directory:
   ```bash
   cd venigem-tech
   ```
3. Open `index.html` in your preferred modern web browser or use a local development server like VS Code's Live Server.

## 🌍 Deployment

This repository is optimized to be deployed seamlessly to [GitHub Pages](https://pages.github.com), Netlify, or Vercel out of the box.
